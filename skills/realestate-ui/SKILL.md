# Skill: Real Estate Listing UI

This project is a real estate platform focused on rental listings.

The UI should help users quickly understand properties and compare options.

---

## Listing UX priorities

Users typically care about:

1. location
2. monthly rent
3. deposit
4. room structure
5. building condition
6. options and amenities
7. parking availability

Important information must be visible immediately.

Avoid forcing users to open multiple screens to see essential details.

---

## Listing card design

Listing cards should be:

- easy to scan
- visually calm
- structured

Prefer layout order:

1. property image
2. price
3. location
4. structure
5. key features

Avoid cluttering listing cards with too many badges or labels.

---

## Comparison usability

The platform should help users compare listings quickly.

Prefer:

- consistent card layout
- consistent information order
- clear price formatting

Avoid layouts where information positions constantly change.

---

## Trust

Real estate services must feel trustworthy.

Avoid:

- flashy visuals
- excessive animations
- overly playful elements

Prefer:

- calm colors
- clear typography
- balanced layouts
