# Skill: Interaction Design

Interactions should feel smooth and natural.

Inspired by:
- Toss
- Airbnb
- modern mobile apps

---

## Motion

Motion should be subtle and purposeful.

Prefer:
- gentle transitions
- smooth page changes
- soft hover effects

Avoid:
- flashy animations
- aggressive motion

---

## Forms

Forms should reduce psychological resistance.

Prefer:

- step-by-step input
- progressive forms
- single-task screens

Avoid:

- long traditional forms
- too many visible inputs

---

## Buttons

Buttons should be clear and easy to understand.

Primary actions should be obvious.

Avoid:
- too many primary buttons
- unclear button hierarchy

---

## Touch friendliness

Mobile usability is critical.

Prefer:
- large tap targets
- comfortable spacing
- thumb-friendly layouts

Avoid:
- small buttons
- dense clickable areas
